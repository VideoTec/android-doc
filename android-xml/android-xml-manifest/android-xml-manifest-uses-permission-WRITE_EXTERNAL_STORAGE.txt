public static final String WRITE_EXTERNAL_STORAGE

Added in API level 4
Allows an application to write to external storage.

Note: If both your minSdkVersion and targetSdkVersion values are set to 3 or lower, the system implicitly grants your app this permission. If you don't need this permission, be sure your targetSdkVersion is 4 or higher.

Starting in API level 19（4.4 KitKat）, this permission is not required to read/write files in your application-specific directories returned by getExternalFilesDir(String) and getExternalCacheDir().

Protection level: dangerous

Constant Value: "android.permission.WRITE_EXTERNAL_STORAGE"
