----------------------------------------------
v17 Leanback Library
----------------------------------------------
The android.support.v17.leanback package provides APIs 
to support building user interfaces on TV devices. 
It provides a number of important widgets for TV apps. 

YouTube Leanback
旨在向用户提供可自定义的、简洁的视频观看体验，
用户可通过其出色的界面观看全高清视频。
此外，沃克还介绍称，YouTube将提供一个Feed源，
向用户推荐他们可能感兴趣的内容。该
服务将在数周内发布测试版

YouTube Leanback脱胎于YouTube XL，
后者于2009年6月发布。
YouTube XL的用户界面比常规YouTube网站更加简洁，
即使隔着一定距离也易于使用，
但用户认为YouTube XL的体验与电视相去甚远。
YouTube Leanback的出现将改变这一看法

com.android.support:leanback-v17:23.1.0
