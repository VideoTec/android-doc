The Android lint tool is a static code analysis tool 
that checks your Android project source files 
for potential bugs and optimization improvements for 
1）correctness, 
2）security, 
3）performance, 
4）usability, 
5）accessibility, and 
6）internationalization.

In Android Studio, 
the configured lint and other IDE inspections run automatically 
whenever you compile your program.

You can also manually run inspections in Android Studio by selecting 
Analyze > Inspect Code from the application or right-click menu. 
The Specify Inspections Scope dialog appears 
so you can specify the desired inspection profile and scope.
