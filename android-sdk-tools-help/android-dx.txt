Android 的 Dalvik VM 是不认 Java 的byte code 的，要用 dx 工具转成Dalvik byte code
当然，dx 工具转了之后，jar 包里面就不是 .class 文件了，而是 .dex 文件

Android DEX文件 ：Android平台上可执行文件的类型 

编译时使用Sun JDK将Java源程序编程成标准的Java字节码文件（.class文件）
通过工具软件DX把所有的字节码文件转成Android DEX文件（classes.dex）
用Android打包工具（aapt）将
1）DEX文件，
2）资源文件以及
3）AndroidManifest.xml文件（二进制格式）组合成一个应用程序包（APK）

dx --dex --output=dexed.jar hello.jar
